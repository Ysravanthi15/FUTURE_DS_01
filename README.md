# 📊 E-Commerce Sales Dashboard (Power BI)

## 🚀 Project Overview

This project presents an interactive **E-Commerce Sales Dashboard** developed using Power BI. It provides a comprehensive analysis of sales performance, customer behavior, and revenue trends through visually appealing and dynamic reports.

The dashboard enables users to explore key business insights and supports data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales performance and trends 📈
* Track key performance indicators (KPIs) 💡
* Identify top-performing products and customers 🏆
* Visualize geographic distribution of sales 🌍
* Enable interactive filtering by country and year 🔍

---

## 📌 Key Metrics

* 💰 **Total Revenue:** 974.8K
* 📦 **Total Orders:** 26K
* 🛒 **Average Order Value:** 376.36
* 👥 **Total Customers:** 406.83K

---

## 📊 Dashboard Features

* KPI cards for quick performance overview
* Monthly revenue trend analysis 📈
* Top product/category contribution analysis
* Customer-wise revenue distribution
* Geographic sales visualization (map view) 🌍
* Interactive filters (Country & Year) for dynamic insights

---

## 🔍 Key Insights

* Strong overall revenue generation with consistent order volume 💰
* Revenue trend shows fluctuations, indicating seasonal patterns 📉📈
* A small number of products contribute significantly to total revenue 🏆
* Customer spending varies, with a few high-value customers driving revenue
* Sales are concentrated in specific regions, especially Europe 🌍
